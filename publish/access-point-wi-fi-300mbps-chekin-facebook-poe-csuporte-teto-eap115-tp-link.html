# Descrição do produto

O Software do Controlador Auranet Gratuito permite administradores a controlar facilmente centenas de EAPs. O Modo Cluster permite gerenciar até 24 (EAP115) sem a necessidade de qualquer dispositivo ou software extra. Compatível com PoE (802.3af) para instalação conveniente e acessível.

- Marca: TP-Link
- Modelo: EAP115
 
Especificações:
 
Hardware:
- Interface: 1 Porta Fast Ethernet (RJ-45) (Suporta PoE IEEE802.3af)
- Botões: Reset
- Fonte de Alimentação: Fornecimento de Energia PoE ou Fonte Externa de 12VDC/1.5A
- Consumo de Energia: 5W
- Padrões Wireless: IEEE 802.11n, IEEE 802.11g, IEEE 802.11b
- Dimensões: 180 x 180 x 47.5mm
- Tipo de Antena: 2 Antenas Internas 3dBi Omni
- Montagem: Montável em Teto/Parede (kit de montagem incluído)
- Trava de Segurança Física: Slot para Trava de Segurança Kensington
- Watch Dog
 
Wireless:
- Frequência: 2.4-2.4835GHz
 
Padrões Wireless:
- IEEE 802.11n
- IEEE 802.11g
- IEEE 802.11b
 
Taxa de Sinal 11n:
- 11n: Até 300Mbps (dinâmico)
- 11g: Até 54Mbps (dinâmico)
- 11b: Até 11Mbps (dinâmico) 
 
Sensibilidade de Recepção:

- 54M: -78dBm@10% PER
- 11M: -93dBm@8% PER
- 6M: -92dBm@10% PER
- 1M: -96dBm@8% PER

- Habilitar/Desabilitar Rádio Wireless

- Controle de Poder de Transmissão（Ajuste de Poder de Transmissão Habilitado em dBm）

- Reinicialização Agendada
- Agendamento Wireless
- Estatísticas Wireless baseadas em SSID/AP/Cliente
 
Segurança Wireless:

- SSID para Mapeamento VLAN 
- Detecção Rogue AP
